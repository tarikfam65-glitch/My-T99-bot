# بوت ANONYMOUS_T11

بوت تيليجرام متكامل مع خدمات ذكاء اصطناعي، فحص ثغرات، تحويل نصوص، وغيرها.

## متطلبات التشغيل

- حساب على Render أو أي منصة تدعم Docker.
- متغيرات البيئة المطلوبة (انظر الأسفل).

## المتغيرات البيئية

أنشئ ملف `.env` أو أضفها في لوحة التحكم:

```env
TELEGRAM_TOKEN=توكن_البوت
ADMIN_ID=معرف_المطور
SERVER_URL=رابط_السيرفر
CLIENT_SECRET_KEY=مفتاح_سري_عشوائي
